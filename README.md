# Лабораторна робота №6
**Варіант:** 8 ("Вгадай число")

## Опис завдання
Програма реалізує гру "Вгадай число" з використанням механізму сопрограм (C++20 Coroutines). 
1. **Сопрограма** загадує ціле число від 1 до 100.
2. **Клієнт (main)** вводить варіанти чисел.
3. Сопрограма порівнює введене число із загаданим і повертає результат:
   - `-1`: число користувача менше загаданого.
   - `1`: число користувача більше загаданого.
   - `0`: число вгадано.
